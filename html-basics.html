HTML Basics: Essential Tags Explained

Document Structure Tags

Every HTML document starts with these essential structure tags:

  • <!DOCTYPE html> - Declares this as an HTML5 document
  • <html> - Root element that wraps all content
  • <head> - Contains metadata (not visible on page)
  • <body> - Contains all visible page content

Text and Heading Tags

HTML provides six levels of headings:

<h1>This is an H1 heading (largest)</h1>

<h2>This is an H2 heading</h2>

<h3>This is an H3 heading</h3>

<h4>This is an H4 heading</h4>

<h5>This is an H5 heading</h5>
<h6>This is an H6 heading (smallest)</h6>

<p>This is a paragraph tag. It's used for regular text content and automatically adds spacing above and below.</p>

Text Formatting Tags

Here are common ways to format text:

  • <strong>Bold text</strong> using <strong> tag
  • <em>Italic text</em> using <em> tag
  • <u>Underlined text</u> using <u> tag
  • <mark>Highlighted text</mark> using <mark> tag
  • <small>Small text</small> using <small> tag

List Tags

Unordered List (bullets)

Syntax:

<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

Renders as:

  • First item
  • Second item
  • Third item

Ordered List (numbers)

Syntax:

<ol>
  <li>First step</li>
  <li>Second step</li>
  <li>Third step</li>
</ol>

Renders as:

  1. First step
  2. Second step
  3. Third step

Block vs Inline Elements

Block Elements (div examples)

Syntax:

<div style="background-color: lightblue; padding: 10px; margin: 5px;">
  This is a div (block element).
</div>

<div style="background-color: lightgreen; padding: 10px; margin: 5px;">
  This is another div.
</div>

Renders as:

Block elements take up the full width available and start on a new line:

This is a div (block element). It takes up the full width and starts on a new line.
This is another div. Notice how it appears below the previous div, even though there's space.
Even when we set width to 50%, this div still starts on a new line.

Inline Elements (span examples)

Syntax:

<p>
  This is a paragraph with
  <span style="background-color: lightcoral; padding: 5px;">inline span elements</span>
  that flow
  <span style="background-color: lightpink; padding: 5px;">within the text</span>
  and don't break to new lines.
</p>

Renders as:

Inline elements only take up as much width as needed and don't start on a new line:

This is a paragraph with inline span elements that flow within the text and don't break to new lines.

Mixed Example

Syntax:

<div style="background-color: #f0f0f0; padding: 10px; margin: 10px;">
  This div contains text and
  <span style="background-color: yellow;">this span stays inline</span>
  with the rest of the text in the div.
</div>

<div style="background-color: #e0e0e0; padding: 10px; margin: 10px;">
  <span style="background-color: orange;">Multiple</span>
  <span style="background-color: cyan;">spans</span>
  <span style="background-color: pink;">in a row</span>
  stay on the same line within this block div.
</div>

Renders as:

Here's what happens when we mix block and inline elements:

This div contains text and this span stays inline with the rest of the text in the div.
Multiple spans in a row stay on the same line within this block div.

Key Differences Summary

  • Block elements (div): Take full width, start on new line, can have width/height set
  • Inline elements (span): Only take needed width, flow with text, width/height ignored

Special HTML Elements

Horizontal Rule (<hr>)

What it does: Creates a horizontal line across the page to separate content sections.

Syntax:

<p>Content above the line</p>
<hr>
<p>Content below the line</p>

Renders as:

Content above the line

Content below the line

Line Break (<br>)

What it does: Forces text to move to the next line without creating a new paragraph.

Syntax:

<p>First line<br>
Second line<br>
Third line</p>

Renders as:

First line
Second line
Third line

Subscript (<sub>) and Superscript (<sup>)

What they do: Make text appear smaller and either below (subscript) or above (superscript) the normal text line. Perfect for chemical formulas, math equations, and footnotes.

Syntax:

<p>Water formula: H<sub>2</sub>O</p>
<p>Einstein's equation: E = mc<sup>2</sup></p>
<p>Mathematical expression: x<sup>2</sup> + y<sub>1</sub> = z<sup>3</sup></p>

Renders as:

Water formula: H2O

Einstein's equation: E = mc2

Mathematical expression: x2 + y1 = z3

Key Points

  • <hr>: Self-closing tag that creates a horizontal line (block element) - takes up full width and starts on new line
  • <br>: Self-closing tag that creates a line break (inline element) - just breaks the line, doesn't add extra spacing
  • <sub>: Makes text appear below the baseline (inline element) - flows with surrounding text
  • <sup>: Makes text appear above the baseline (inline element) - flows with surrounding text
