HTML Tables: Complete Guide

What are HTML Tables?

HTML tables are used to display data in rows and columns, similar to a spreadsheet. They're perfect for presenting structured information like schedules, price lists, statistics, or any data that benefits from a grid layout.

Basic Table Structure

Every HTML table is built using these essential tags:

  • <table> - The container for the entire table
  • <tr> - Table row (horizontal)
  • <td> - Table data cell (regular cell)
  • <th> - Table header cell (bold and centered by default)

Simple Table Example

Code:

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Sarah</td>
    <td>30</td>
    <td>London</td>
  </tr>
</table>

Renders as:

Name Age City
John 25 New York
Sarah 30 London

Table Structure Tags

For better organization and accessibility, tables can be divided into sections:

  • <thead> - Groups header content
  • <tbody> - Groups body content
  • <tfoot> - Groups footer content

Structured Table Example

Code:

<table border="1">
  <thead>
    <tr>
      <th>Product</th>
      <th>Price</th>
      <th>Quantity</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Laptop</td>
      <td>$999</td>
      <td>5</td>
    </tr>
    <tr>
      <td>Mouse</td>
      <td>$25</td>
      <td>20</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td><strong>Total Items</strong></td>
      <td><strong>$1024</strong></td>
      <td><strong>25</strong></td>
    </tr>
  </tfoot>
</table>

Renders as:

Product Price Quantity
Laptop $999 5
Mouse $25 20
Total Items $1024 25

Spanning Cells

You can make cells span multiple rows or columns:

  • colspan - Makes a cell span multiple columns
  • rowspan - Makes a cell span multiple rows

Colspan Example

Code:

<table border="1">
  <tr>
    <th colspan="3">Monthly Sales Report</th>
  </tr>
  <tr>
    <th>Month</th>
    <th>Sales</th>
    <th>Target</th>
  </tr>
  <tr>
    <td>January</td>
    <td>$5000</td>
    <td>$4500</td>
  </tr>
</table>

Renders as:

Monthly Sales Report
Month Sales Target
January $5000 $4500

Rowspan Example

Code:

<table border="1">
  <tr>
    <th>Department</th>
    <th>Employee</th>
    <th>Role</th>
  </tr>
  <tr>
    <td rowspan="2">IT</td>
    <td>Alice</td>
    <td>Developer</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>Designer</td>
  </tr>
</table>

Renders as:

Department Employee Role
IT Alice Developer
Bob Designer

Table Caption

Use the <caption> tag to add a title or description to your table:

Code:

<table border="1">
  <caption>Student Grades - Fall 2024</caption>
  <tr>
    <th>Student</th>
    <th>Grade</th>
  </tr>
  <tr>
    <td>Emma</td>
    <td>A</td>
  </tr>
</table>

Renders as:

Student Grades - Fall 2024
Student Grade
Emma A

Best Practices

  • Use tables for tabular data only - Don't use tables for page layout
  • Always include headers - Use <th> tags for better accessibility
  • Add a caption - Helps users understand the table's purpose
  • Use semantic structure - Include <thead>, <tbody>, and <tfoot> when appropriate
  • Style with CSS - Use CSS instead of HTML attributes for styling
