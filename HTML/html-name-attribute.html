HTML Name Attribute: Complete Guide

What is the Name Attribute?

The name attribute is used to specify a name for an HTML element, particularly form elements. It's essential for:

  • Identifying form data when submitted to a server
  • Referencing form elements in JavaScript
  • Grouping radio buttons together

Common Uses of Name Attribute

1. Text Input Fields

Example:

Code:

<input type="text" name="username" placeholder="Enter username">

2. Radio Buttons

Example:

Male Female

Code:

<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female

3. Checkboxes

Example:

Reading Gaming

Code:

<input type="checkbox" name="hobbies[]" value="reading"> Reading
<input type="checkbox" name="hobbies[]" value="gaming"> Gaming

4. Select Dropdown

Example:

Code:

<select name="country">
  <option value="usa">USA</option>
  <option value="uk">UK</option>
  <option value="canada">Canada</option>
</select>

Form Example with Multiple Name Attributes







Note: When a form is submitted, the name attribute is used to identify each form control in the submitted data. The server receives the data as name/value pairs.

YouTube Search Form Example

Example:

Code:

<form action="https://www.youtube.com/results" method="get">
  <label>
    Search YouTube:
    <input type="search" name="search_query" placeholder="Enter your search...">
  </label>
  <button type="submit">Search YouTube</button>
</form>
